# Blood Bowl rules reference

The rules of **Blood Bowl: Third Season Edition (2025)** as the project's working
reference — the source for the in-app codex and tooltips, and what the engine and
data are built against. Mechanics are summarised with the factual data (target
rolls, modifiers, costs, tables); each file ends with an **Implementation status**
note so the gap between "documented" and "modelled" stays visible.
