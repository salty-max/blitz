# Blood Bowl rules reference

These notes cover **Blood Bowl: Third Season Edition (2025)**. They restate the
rules **in our own words** alongside the factual data (target rolls, modifiers,
costs, tables) — they are **not** a copy of the rulebook. They are the source for
the in-app codex and tooltips, and the reference the engine/data are built
against.

## Conventions

- **Original wording.** Each mechanic is summarised in our own words. Short
  quotes only where the exact wording is load-bearing, and then attributed.
- **Facts are listed plainly** — `X+` targets, modifiers, costs, and tables.
  Game facts aren't copyrightable; GW's prose is, so we don't reproduce it.
- **Sources** are listed at the foot of each file. The official Third Season
  Edition rulebook is the authority; community transcriptions (e.g.
  bloodbowlbase.ru) are a convenience and should be verified against it.
- **Implementation status** per file records what the engine/data currently
  cover, so the gap between "documented" and "modelled" is always visible.
