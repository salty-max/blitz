# Skills & traits

Players gain **Skills** through advancement (grouped into categories a position
can access as Primary or Secondary — see [team-building.md](team-building.md)),
and have innate **Traits**. This is the catalogue, with a short description of
each effect.

## General

| Skill          | Effect                                                      |
| -------------- | ----------------------------------------------------------- |
| Block          | not knocked down on a Both Down result                      |
| Dauntless      | roll to match a stronger opponent's Strength for a block    |
| Fend           | opponents can't follow up after pushing you                 |
| Frenzy         | must follow up and throw a second block if the target moved |
| Kick           | reduce kick-off deviation to a D3                           |
| Pro            | once per activation, roll 3+ to re-roll one die             |
| Steady Footing | roll a 6 to avoid a knock-down / fall                       |
| Strip Ball     | a ball carrier you push back drops the ball                 |
| Sure Hands     | re-roll a failed pick-up; immune to Strip Ball              |
| Tackle         | opponents can't Dodge out of your tackle zone               |
| Taunt          | force a player you pushed back to follow up                 |
| Wrestle        | on a Both Down, put both players prone                      |

| Sidestep           | choose your own push-back square                          |

| Sure Feet          | once per turn, re-roll a failed Rush                      |
