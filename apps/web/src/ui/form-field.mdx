# FormField

Wraps a single form control with a label and an optional hint or error. It
generates an `id`, links the label to the control, and wires `aria-describedby`
and `aria-invalid` to the message — so dropping any control inside (`Input`,
`Switch`, `Select`…) gets you an accessible field for free.

## Hint and error

The `error` message replaces the `hint` and flags the control invalid; `required`
adds a blood asterisk to the label.
