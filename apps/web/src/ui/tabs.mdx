# Tabs

Page-level tabbed sections — Radix-powered, with arrow-key navigation. The active
trigger underlines blood. Compound parts: `Tabs.List`, `Tabs.Trigger` and
`Tabs.Content`.

Use it to split a page into sections — a team page (Roster / Staff / History), a
league page (Standings / Fixtures / Stats). For a small inline choice, reach for
`SegmentedControl` instead.
