# Table

A horizontally scrollable data table with the codex's heavy header rule.
Assembled from compound parts: `Table.Header`, `Table.Body`, `Table.Row`,
`Table.Head` and `Table.Cell`. Rows are baseline-aligned so a tall cell lines up
with its roll.
