# SegmentedControl

A single-select control rendered as a row of equal-width segments with a sliding
ink indicator behind the active one. Built on Radix's toggle group, so it has
radio-group semantics and arrow-key navigation; the indicator is exactly one
segment wide and slides by whole segments, so it always covers the target.
