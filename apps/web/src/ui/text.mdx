# Text

The single source of typographic truth. Every run of styled text flows through
`Text` so faces, scale and colour stay consistent. Pick the semantic tag with
`as` (or `asChild` to style an existing child, like a router `Link`), and the
look with `variant` and `tone`; `weight` and `tabular` are orthogonal overrides
for the cases a variant's defaults don't cover.

## The scale

Every `variant`, from the heavy display face down to fine print. The heading
primitives (`PageHeading`, `SectionHeading`, `Eyebrow`) and `EmptyState` /
`StatBlock` are thin wrappers over these.
