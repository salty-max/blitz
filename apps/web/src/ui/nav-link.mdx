# NavLink

Class variants for a navigation link, not a component. Active styling keys off
the router's `data-status="active"` attribute, so apply `navLinkVariants(...)`
to a router `Link`'s `className` — no active/inactive props needed. Two
contexts ship: the codex `section` nav and the `masthead`.
