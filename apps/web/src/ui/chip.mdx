# Chip

A small uppercase headline pill — a static label, a price, or an interactive
reference tag.

## Filled variants

`blood`, `ink` and `pitch` colour-code the reference drawer's kind badge;
`gold` (with `tabular-nums`) is used for cost badges.

## Bordered, interactive

`outline` and `accent` fill with blood on hover — used for the reference and
team chips. Combine with `asChild` to render a link or button as the chip.
