# Combobox

A searchable single-select — a trigger that opens a filtered list in a popover.
Reach for it over `Select` when the options are many: hiring a star player,
picking a team or a skill. It's controlled: pass `options`, the selected `value`
(or `null`), and an `onValueChange`.
