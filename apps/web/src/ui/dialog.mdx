# Dialog

A centered modal — Radix Dialog styled as a bordered paper card with a hard
offset shadow. Compound parts: `Dialog.Trigger`, `Dialog.Content`,
`Dialog.Title`, `Dialog.Description`, `Dialog.Footer` and `Dialog.Close`.

Use it for confirmations (disbanding a team, deleting a result) and short modal
forms. Wrap an action button in `Dialog.Close` with `asChild` to confirm and
dismiss in one click.
