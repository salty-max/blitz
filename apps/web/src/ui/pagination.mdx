# Pagination

A page navigator — previous/next arrows around a windowed list of page numbers,
the active page filled blood. Long star-player or fixture lists, anything paged.
It's controlled: pass the 1-based `page`, the `pageCount`, and an `onPageChange`.

## Few pages

With seven pages or fewer, every page shows; beyond that, it collapses the middle
to an ellipsis around the current page.
