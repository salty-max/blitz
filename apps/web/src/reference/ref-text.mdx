# RefText

Renders text containing `[[key]]` / `[[key|label]]` references, turning each into
a button that opens it in the side-drawer — colour-coded by the kind of entry it
points to (skill, rules term, special rule, star ability). Plain text passes
through unchanged. It must be rendered inside a `RefDrawerProvider`.
